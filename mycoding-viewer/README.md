# mycoding-viewer

## Description 
> **front-back stage decoupling framework**  
> **GitHub: [https://github.com/csulwk/mycoding.git](https://github.com/csulwk/mycoding.git)**  

## Environment
* `node 12.16.1`  

## Framework
```
mycoding-viewer  
├─ build                                    -- 构建脚本
├─ config                                   -- 项目配置
├─ src                                      -- 源码目录
├─ static                                   -- 静态资源
├─ test                                     -- 测试文件
├─ index.html                               -- 入口页面
├─ .gitignore
└─ README.md
```

## Setup
```
npm install
npm run serve
npm run build
```
